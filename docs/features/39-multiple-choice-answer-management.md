# Feature 39: Multiple Choice Answer Management

**Status**: Pending  
**Priority**: Medium  
**Category**: Frontend/Admin

## Description

Implement interface for managing multiple choice answer options, including adding, editing, deleting, and reordering answers.

## Requirements

- Display list of answer options
- Add new answer option
- Edit existing answer option
- Delete answer option
- Mark correct answer(s)
- Reorder answers (drag and drop or up/down buttons)
- Validate at least one correct answer

## Acceptance Criteria

- [ ] Answer options displayed in order
- [ ] Add answer button creates new option
- [ ] Edit answer updates option
- [ ] Delete answer removes option
- [ ] Correct answer can be marked
- [ ] Answers can be reordered
- [ ] Validation ensures at least one correct answer
- [ ] Order field updated when reordering

## Technical Notes

- Create answer management component
- Use drag-and-drop library (optional) or up/down buttons
- Update order field in database
- Follow form patterns
