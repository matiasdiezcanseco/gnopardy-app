# Feature 24: Player List Component

**Status**: Pending  
**Priority**: Medium  
**Category**: Frontend/Components

## Description

Create a PlayerList component that displays all players in the current game with their names and avatars, allowing selection.

## Requirements

- Display all players in current game
- Show player names
- Display player avatars (if available)
- Allow player selection
- Highlight selected player
- Show player scores (optional)

## Acceptance Criteria

- [ ] PlayerList displays all players
- [ ] Player names shown clearly
- [ ] Avatars displayed (or initials if no avatar)
- [ ] Players are clickable for selection
- [ ] Selected player highlighted
- [ ] Component updates when players added/removed
- [ ] Responsive layout

## Technical Notes

- Create `src/components/player/PlayerList.tsx`
- Use shadcn/ui Avatar component
- Integrate with player selection context
- Follow component patterns
